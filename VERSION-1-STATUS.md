# The Pack Version 1

## Included in this build

- Five-tab bottom navigation: Home, Events, Journal, Friends, Profile
- Full-screen single-page journal
- Leather cover as the Journal landing screen
- First open goes to the Explorer's Guide
- Later visits return to the last interior page viewed
- Cover, Explorer's Guide, live Table of Contents, and all blank collection pages
- Three user-selected sticker positions per collection page
- Admin sticker upload, section assignment, code assignment, activation, and deletion
- Staff-code sticker unlocking and reveal
- Friends currently checked in by location (demo data)
- Supabase starter schema and setup guide

## Current demo storage

The front end currently uses browser local storage. Data does not sync between devices or users until Supabase is connected.
